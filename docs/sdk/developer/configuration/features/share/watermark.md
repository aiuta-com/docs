# [:material-arrow-up-left:](index.md#share) Watermark

Optional configuration for adding a watermark to shared content.
```typescript
ShareWatermarkFeature {
  images {
    logo: Image // (1)!
  }
}
```

1.  Logo image to be used as a watermark on shared content.
